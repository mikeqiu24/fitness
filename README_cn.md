基于RDK X5的主动式AI健身机器人

摘要

针对当前家庭智能健身领域缺少主动式、个性化智能陪练设备，传统健身指导存在人工成本高、实时纠错能力不足、无法动态跟随适配等问题，本文设计并实现了一套主动式AI健身陪练机器人系统，可适配大众居家健身、肢体康复训练、智能人机交互等多元应用场景。系统采用分层模块化架构，以RDK X5边缘AI开发板作为视觉感知与智能决策核心，结合STM32F103C8T6单片机作为运动控制执行单元，构建了视觉感知、智能决策、运动执行、云端数据同步的完整闭环体系，兼顾识别精度、运动稳定性与数据实时性。系统搭载单目MIPI摄像头实时采集人体健身图像，依托板载BPU神经网络加速器部署轻量化YOLO11-Pose量化模型，完成17点人体骨骼关键点精准检测，运行帧率可达15FPS以上，满足健身姿态实时检测的动态性能需求。系统基于骨骼关键点坐标解算人体关节角度与肢体运动幅度，针对深蹲等典型健身动作建立量化评分标准，可自动评判动作标准度、识别姿态偏差，并输出实时纠错提示，有效解决用户自主健身动作不规范、缺乏专业指导的痛点。在人机协同运动方面，系统融合单目深度估计与人体运动轨迹分析算法，精准捕捉用户位置变化与运动意图，驱动麦克纳姆轮全向底盘实现主动跟随功能。底盘采用三自由度运动学解算与PWM平滑过渡算法，可稳定完成前后移动、左右平移、原地旋转等复合运动，有效消除启停与转向冲击，运动过程平稳灵敏。通信层面，RDK X5与STM32单片机通过UART串口实现指令应答通信，采用帧头帧尾标识与异或校验机制，大幅降低数据丢包、误包概率，保障板间通信稳定可靠。同时，系统通过WiFi模块将健身评分、运动轨迹、训练时长等数据实时上传至OneNET物联网云平台，实现健身数据远程监控、云端留存与历史追溯，为用户健身规划和康复效果评估提供数据支撑。实测结果表明，该系统识别精度高、响应速度快、运行稳定性强，人机交互体验良好，能够有效满足居家智能健身与康复训练的指导需求，具备较高的实用价值与广阔的市场应用前景。

 



第一部分 作品概述

1.1功能与特性

本系统具备三大核心功能：一是单目视觉人体姿态实时识别功能，可准确检测人体17个骨骼关键点坐标；二是智能动作指导功能，基于关节角度计算实现动作标准度评分与实时纠错提示；三是云端数据同步功能，运动数据实时上传至云平台，支持运动记录追溯与数据分析。系统支持OLED实时状态显示，具备完善的异常处理与通信可靠性机制。

1.2应用领域

本作品主要面向三大应用场景：一是家庭智能健身领域，可作为私人健身陪练，实时纠正用户动作姿势，提升居家健身效果，避免错误动作导致的运动损伤；二是运动康复领域，可在医生指导下辅助患者进行康复训练，量化评估康复进度；三是智能人机交互领域，全向移动底盘结合人体姿态识别可应用于服务机器人、智能跟随设备、人机协作等场景。此外，本系统的分层架构设计也可作为嵌入式AI教学与竞赛开发平台。

1.3主要技术特点

（1）端侧AI推理：采用RDK X5的10TOPS BPU硬件加速，实现人体姿态识别的端侧实时推理，无需依赖云端，延迟低隐私性好；

（2）全向移动底盘：麦克纳姆轮运动学解算实现三自由度复合运动，配合PWM斜坡算法实现平滑加减速；

（3）高可靠通信：UART串口采用帧头帧尾加异或校验的指令-应答机制，误码率低于0.1%；

（4）多模态闭环：视觉感知、决策控制、运动执行、云端同步形成完整闭环系统。

1.4主要性能指标

主要性能指标表：

| **性能指标**       | **参数值**          |
| ------------------ | ------------------- |
| 人体姿态检测帧率   | ≥15 FPS             |
| 关键点检测准确率   | ≥90%                |
| 底盘运动自由度     | 3自由度（X/Y/旋转） |
| 指令响应延迟       | ≤100 ms             |
| 云端数据上报成功率 | ≥99%                |

1.5主要创新点

（1）将边缘AI姿态识别与麦克纳姆轮全向移动结合，实现主动式健身陪练而非被动监测；

（2）基于关节角度的动作标准度量化评估算法，提供实时纠错反馈；

（3）单目深度与轨迹分析结合的人体跟随算法，实现自然流畅的人机交互；

（4）分层解耦的软硬件架构，视觉、控制、通信、云端各模块独立可扩展。

1.6设计流程

本作品采用模块化迭代开发流程：首先进行系统需求分析与总体架构设计，确定三层架构方案；然后分模块并行开发，包括RDK X5视觉算法开发、STM32底盘控制开发、云平台对接开发；各模块单元测试通过后进行系统集成与联调，优化通信协议与控制参数；最后进行整机性能测试与稳定性优化，完善异常处理机制。开发过程中采用软硬件协同设计方法，在硬件设计阶段就考虑软件驱动与接口协议，缩短了整体开发周期。

第二部分 系统组成及功能说明

2.1整体介绍

系统采用三级分层架构：感知决策层以RDK X5为核心，负责单目图像采集、BPU姿态推理、动作评估、指令生成与云端通信；执行控制层以STM32F103C8T6为核心，负责串口指令解析、电机运动控制与状态反馈；云端服务层采用OneNET物联网平台，负责数据存储与远程监控。两层主控之间通过UART串口以115200bps波特率通信，采用自定义的可靠通信协议。系统整体供电采用7.4V锂电池组，经稳压模块分别为RDK X5（5V/3A）、STM32与电机驱动供电，电源之间做隔离设计避免干扰。



**图2-1****系统整体框图**

2.2硬件系统介绍

2.2.1 硬件整体介绍

硬件系统主要由RDK X5 AI计算板、御光-SC132GS单目摄像头模组、STM32F103C8T6最小系统板、四路直流电机驱动板、麦克纳姆轮底盘总成、0.96寸OLED显示屏、7.4V锂电池组与稳压模块组成。RDK X5通过两路MIPI CSI接口连接单目摄像头，通过40Pin排针的UART1连接STM32的USART1，注意TX与RX需要交叉连接且必须共地。STM32通过TIM2定时器输出4路PWM信号控制电机转速，通过GPIO控制电机方向，通过软件I2C连接OLED显示屏。

2.2.2 核心电路模块介绍

电机驱动采用H桥电路，每路电机通过两个GPIO控制方向，一个PWM引脚控制转速。H桥驱动逻辑为：IN1=0,IN2=1正转；IN1=1,IN2=0反转；IN1=IN2刹车。电源部分采用LM2596开关稳压模块将电池电压降至5V为RDK X5供电，AMS1117-3.3为STM32与OLED供电，电机电源与逻辑电源之间采用磁珠隔离。串口通信线上串联33Ω电阻做阻抗匹配，减少信号反射。



2.3软件系统介绍

2.3.1 软件整体介绍

软件系统分为RDK X5上位机软件与STM32下位机软件两部分。RDK X5运行RDK OS 3.4.1系统，基于TogetheROS.Bot机器人中间件开发，主要包含单目图像采集、BPU姿态推理、动作标准度评估、运动轨迹解析、UART通信、MQTT云端上传六个模块。STM32基于HAL库开发，采用中断+状态机架构，主要包含串口帧解析、电机运动控制、PWM平滑过渡、OLED显示四个模块。软件整体采用分层设计，模块之间通过标准接口调用，耦合度低便于维护扩展。



**图2-4****软件架构图**



2.3.2 核心算法模块介绍

（1）人体姿态检测算法：采用YOLO11-Pose量化模型，通过BPU加速推理，输出17个COCO标准关键点坐标与置信度，低置信度关键点采用一阶滞后滤波平滑处理。

（2）动作标准度评估算法：以关节点为中心，通过向量点积计算相邻骨骼夹角，与标准动作阈值对比，输出0-100分评分与具体纠错提示。

（3）麦克纳姆轮运动学算法：三速度分量（Vx前后/Vy左右/Rot旋转）解算四个电机转速，公式为：左前=Vx-Vy-Rot，右前=Vx+Vy+Rot，左后=Vx+Vy-Rot，右后=Vx-Vy+Rot。

（4）PWM平滑过渡算法：采用斜坡函数，每10msPWM变化5%，从0到满速加速时间200ms，避免电流尖峰与机械冲击。

（5）串口解析状态机：6状态有限状态机逐字节解析，包含帧头校验、长度计数、异或校验、帧尾校验，有效解决粘包半包问题。

2.3.3 通信协议设计

本系统STM32主控与上位机、遥控终端采用USART串口异步通信，为解决裸数据传输易干扰、丢包、解析混乱、无状态反馈等问题，自定义一套标准化固定帧结构通信协议，实现指令下发、设备执行、状态回传的闭环通信，保障设备启停、行走、转向等运动控制的传输稳定性。协议统一帧结构为帧头0xAA、指令码、数据长度、数据域、异或校验和、帧尾0x55，通过指令码最高位区分帧类型，高位为0是上位机控制指令帧，高位为1是STM32应答帧，定义了停止、前进、后退、左转、右转五条基础控制指令。帧结构各字段分工明确，1字节数据长度字段限定0~128字节可变数据域，用于承载速度、角度、时长等控制参数，无参指令数据域为空；采用指令码、数据长度、数据域整体异或校验方式，运算轻量化适配STM32实时运行需求。设备解析指令后会回传状态应答，包含执行成功、未知指令、参数错误、设备忙、设备故障五类状态码。通信过程配备完善异常处理机制，帧格式错误、校验失败、参数异常时自动丢弃数据或返回错误码，协议结构简洁、抗干扰性强，同时预留拓展空间，具备良好实用性与可扩展性。
